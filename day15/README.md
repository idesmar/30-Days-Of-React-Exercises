### Learnings
* css modules
  * Naming
  ```
  moduleName.module.css
  ```
  * Importing
  ```js
  import moduleName from './moduleName.module.css'
  // import altName from './moduleName.module.css'
  ```
  * Destructuring imported style to extract class name/s
  ```js
  const { className1, className2 } = moduleName
  ```
